## Название проектной работы
[Kalyaka](https://kulembetov.github.io/kalyaka)

## Описание
Одностраничный сайт, был свёрстан по макету из Figma и адаптирован под различные устройства (320px, 768px, 1440px). Сделан в рамках учёбной деятельности в [Яндекс.Практикуме](https://practicum.yandex.ru) на факультете ["Веб-разработчик плюс"](https://practicum.yandex.ru/web-plus).
